# Frontend (GitHub Pages)

1) Edit `config.js`:
- BACKEND_URL = your Cloudflare tunnel URL (https://xxxx.trycloudflare.com)
- ADMIN_CODE / PUBLIC_CODE (must match backend env)

2) Upload all files in this folder to GitHub repo root (or /docs).

3) GitHub Pages should serve `index.html` automatically.

Pages:
- /index.html (landing)
- /group.html
- /admin.html
- /public.html
